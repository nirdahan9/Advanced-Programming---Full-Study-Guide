# Unified Study Guides Site

אתר מאוחד לתרגול שאלות מכל המדריכים:
- Git
- Java
- Python
- SQL
- Web

## מה נכלל
- כל השאלות מהאתרים הקיימים.
- עבור Web נכללו שתי קבוצות השאלות (`questions_v2.js` וגם `questions.js`) כדי לוודא כיסוי מלא.
- כל מאגרי השאלות הועתקו לתיקייה המקומית [questions/](questions/) כך שהאתר עצמאי לחלוטין ולא תלוי בתיקיות חיצוניות.

## איך להשתמש
1. פתח/י את `index.html`.
2. בחר/י נושא אחד, כמה נושאים, או הכל.
3. בחר/י מספר שאלות ולחץ/י על התחלה.
